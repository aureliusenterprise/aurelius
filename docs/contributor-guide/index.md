# Contributor Guide

Welcome to the Aurelius Atlas contributor guide!

This guide is designed to help new and existing contributors get up to speed with the project, understand the
development environment, and learn how to contribute effectively.

Whether you're fixing a bug, adding a new feature, or improving documentation, this guide will provide you with
the information you need to get started.

## Table of Contents

This guide covers the following topics:

- [Development Environment](./development-environment.md): Instructions on setting up your local development
  environment.
- [Secrets Management](./secrets-management.md): Best practices for managing sensitive information securely within
  the project repository.
