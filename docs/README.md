# Aurelius Atlas

Welcome to the Aurelius Atlas workspace! This repository contains the source code for the Aurelius Atlas, a metadata
management system built on top of Apache Atlas.

## Getting Started

To get started with the Aurelius Atlas workspace, please refer to the [Contributor Guide](./contributor-guide/index.md)
guide. This guide will walk you through setting up your local development environment, including how to run the
application and its dependencies.
