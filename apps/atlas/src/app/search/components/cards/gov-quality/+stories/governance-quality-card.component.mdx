# GovernanceQualityDetailsCardComponent

This is the details card template for governance quality results, as well as inheriting types.

## Primary

Please find an example of a governance quality details card below.

<Canvas>
  <Story of={GovernanceQualityCardComponentStories.Primary} />
</Canvas>

### Content

A governance quality details card displays the following content:

| Content    | Description                                                                   |
| ---------- | ----------------------------------------------------------------------------- |
| Title      | The name of this entity.                                                      |
| Type       | The type of this entity in the data governance model.                         |
| Compliance | Whether or not the related entity is compliant with this data governance rule |
