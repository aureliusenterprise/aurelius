# FieldDetailsCardComponent

This is the details card template for fields, as well as inheriting types.

## Primary

Please find an example of a field details card below.

<Canvas>
  <Story of={FieldDetailsCardComponentStories.Primary} />

### Content

A field details card displays the following content:

| Content                 | Description                                                                                     |
|-------------------------|-------------------------------------------------------------------------------------------------|
| Title                   | The name of this entity.                                                                        |
| Type                    | The type of this entity in the data governance model.                                           |
| Breadcrumb              | This entity's parents in the data governance model.                                             |
| Classifications         | The classifications assigned or propagated to this entity.                                      |
| Data quality            | The (aggregated) data quality scores for this entity, ranging from 0% to 100%.                  |
| Data governance quality | An indicator of whether or not there are data governance quality issues related to this entity. |
