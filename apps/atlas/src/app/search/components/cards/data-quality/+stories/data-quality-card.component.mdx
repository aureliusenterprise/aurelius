# DataQualityDetailsCardComponent

This is the details card template for data quality results, as well as inheriting types.

## Primary

Please find an example of a data quality details card below.

<Canvas>
  <Story of={DataQualityCardComponentStories.Primary} />
</Canvas>

### Content

A data quality details card displays the following content:

| Content      | Description                                                                    |
| ------------ | ------------------------------------------------------------------------------ |
| Title        | The name of this entity.                                                       |
| Type         | The type of this entity in the data governance model.                          |
| Breadcrumb   | This entity's parents in the data governance model.                            |
| Data quality | The (aggregated) data quality scores for this entity, ranging from 0% to 100%. |
