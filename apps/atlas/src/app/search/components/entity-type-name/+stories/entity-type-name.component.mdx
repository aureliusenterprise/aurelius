# EntityTypeNameComponent

This component displays type names, type aliases and associated icons within the context of an Atlas entity.

<Controls />

## Primary

The `models4insight-entity-type-name` subscribes to the `EntityDetailsService` provided in the context in which the component is used.
It automatically picks up the type name and type alias if available.
It is possible to customize the appearance of the component similar to the `models4insight-type-name` component.
