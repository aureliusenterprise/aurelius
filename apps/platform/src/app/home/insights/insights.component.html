As a decision maker, you have access to various dashboard related to your team's projects.

Set up your account

You can set up your account via the account management page. Here, you have the option to add additonal info, reset your password, and to set up a mobile authenticator for improved security.

Access your team's project

Work in Models4Insight is organized as projects. Your team likely has set up a project already. You need to be granted access to this project before you can view any dashboards.

Please share the information below with your team so they can give you access:

Username

Copy to clipboard

Once you have access to a project, it will be visible on your projects page. Press the button below to go there directly. The projects page is also accessible via the top navigation bar.

Explore models

When in a project, you will be able to explore the model via the Model Explorer. There, you can browse through various aspects of the model that your team has prepared.

To access the Model Explorer, first open your project. Then, find the version of the model that you want to have a look at. The Project Homescreen provides an overview of the most recent versions, while the Retrieve a Model page offers additional search capabilities. You will find the link to the Model Explorer under the Context Menu for every model version.
