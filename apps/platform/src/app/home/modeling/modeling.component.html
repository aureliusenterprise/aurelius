As a modeler, Models4Insight provides you with various tools to maintain, manage and explore your models.

Set up your account

You can set up your account via the account management page. Here, you have the option to add additonal info, reset your password, and to set up a mobile authenticator for improved security.

Create a project or join an existing one

Your work in Models4Insight is organized as projects. Every project encompasses a single model, and provides management functions for that model such as version control and access control. A project owner can invite you to join their project.
